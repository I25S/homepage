3.Systems Development​

i25s has a fine track record of delivering turn-key solutions where customer value is the main measure. We offer full-stack services and are used to work on highly complex and highly integrated enterprise solutions where up-time and performance are a given. The most recent projects have been on both MS Azure and Amazon services, with integrations to both third-party and on-prem systems. Examples are:

  • Personnel overview for on-location technicians with Operator Console and cross-platform technician solution, building on server-less MS Azure services, legacy communication technology and full integration to on-prem Active Directory for major Danish Grid Operator​
  • Android and iOS based application integrated to back-end solution that provides bot-based communication services and more, currently under development and at the Alpha stage​
  • Internal development of a Distributed Database system that allows for robust and secure data handling across multiple nodes being based on patent pending inhouse inventions
